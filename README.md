## 0. 心智模型：标志位与比较

`cmp a, b` 在内部执行 `a - b` 但**只更新标志位、不写回结果**。后续的条件跳转读取这些标志位。理解“有符号”与“无符号”两套跳转的区别是关键——它们读取**不同的标志位组合**。

**核心标志位：**

| 标志 | 名称          | 含义                                  |
| ---- | ------------- | ------------------------------------- |
| `ZF` | Zero Flag     | 结果为 0（即 `a == b`）               |
| `SF` | Sign Flag     | 结果最高位为 1（结果为负）            |
| `CF` | Carry Flag    | 无符号运算发生进位/借位（无符号溢出） |
| `OF` | Overflow Flag | 有符号运算溢出                        |

**有符号 vs 无符号比较的判定逻辑：**

- **无符号** 看 `CF` 和 `ZF`：`a < b` ⟺ `CF=1`（减法借位说明 a 更小）。
- **有符号** 看 `SF` 和 `OF`：`a < b` ⟺ `SF ≠ OF`（异号说明发生了符号溢出，真实大小关系被翻转）。

> 直觉：有符号比较若没溢出，`SF` 直接告诉你正负即大小；一旦溢出（`OF=1`），符号位骗人了，于是用 `SF ≠ OF` 把它纠正回来。

**助记表（你给出的部分 + 补全）：**

| 指令（同义词）        | 条件                | 含义     | 符号性     |
| --------------------- | ------------------- | -------- | ---------- |
| `je` / `jz`           | `ZF=1`              | `a == b` | 通用       |
| `jne` / `jnz`         | `ZF=0`              | `a != b` | 通用       |
| `jl` / `jnge`         | `SF ≠ OF`           | `a < b`  | **有符号** |
| `jge` / `jnl`         | `SF = OF`           | `a >= b` | **有符号** |
| `jle` / `jng`         | `ZF=1` 或 `SF ≠ OF` | `a <= b` | **有符号** |
| `jg` / `jnle`         | `ZF=0` 且 `SF = OF` | `a > b`  | **有符号** |
| `jb` / `jc` / `jnae`  | `CF=1`              | `a < b`  | **无符号** |
| `jae` / `jnc` / `jnb` | `CF=0`              | `a >= b` | **无符号** |
| `jbe` / `jna`         | `CF=1` 或 `ZF=1`    | `a <= b` | **无符号** |
| `ja` / `jnbe`         | `CF=0` 且 `ZF=0`    | `a > b`  | **无符号** |

> 命名规律：有符号用 **L**ess / **G**reater；无符号用 **B**elow / **A**bove。`n` 前缀表示取反（not）。

## 1. 数据传送

| 指令     | 形式                | 说明                                                                                   |
| -------- | ------------------- | -------------------------------------------------------------------------------------- |
| `mov`    | `mov dst, src`      | 传送。不能 mem→mem，不能立即数→段寄存器。                                              |
| `lea`    | `lea r, [addr]`     | 加载**有效地址**（算地址，不访存）。也常用于快速算术，如 `lea rax,[rdi+rdi*2]`（×3）。 |
| `xchg`   | `xchg a, b`         | 交换两操作数。带内存操作数时隐含 `lock`（昂贵）。                                      |
| `movzx`  | `movzx r, r/m`      | 零扩展传送（无符号放大），如 8→64 位高位补 0。                                         |
| `movsx`  | `movsx r, r/m`      | 符号扩展传送（有符号放大），高位补符号位。                                             |
| `movsxd` | `movsxd r64, r/m32` | 32→64 位符号扩展（x86-64 专用）。                                                      |
| `cmovcc` | `cmov<cc> r, r/m`   | 条件传送，见 [§8](#8-条件传送与置位cmovcc--setcc)。                                    |
| `bswap`  | `bswap r32/r64`     | 字节序反转（大小端转换）。                                                             |

> **64 位陷阱**：向 32 位寄存器（如 `eax`）写入会**自动清零**对应 64 位寄存器（`rax`）的高 32 位；但向 8/16 位子寄存器写入则**不会**清零高位。

## 2. 算术运算

| 指令   | 形式                                            | 说明                                            | 影响标志          |
| ------ | ----------------------------------------------- | ----------------------------------------------- | ----------------- |
| `add`  | `add dst, src`                                  | 加法                                            | ZF SF CF OF AF PF |
| `sub`  | `sub dst, src`                                  | 减法                                            | ZF SF CF OF AF PF |
| `adc`  | `adc dst, src`                                  | 带进位加（`dst+src+CF`），用于多精度加法        | 同 add            |
| `sbb`  | `sbb dst, src`                                  | 带借位减（`dst-src-CF`），用于多精度减法        | 同 sub            |
| `inc`  | `inc r/m`                                       | 自增 1。**不影响 CF**（注意！）                 | ZF SF OF AF PF    |
| `dec`  | `dec r/m`                                       | 自减 1。**不影响 CF**                           | ZF SF OF AF PF    |
| `neg`  | `neg r/m`                                       | 取负（二进制补码，`0 - operand`）               | ZF SF CF OF AF PF |
| `mul`  | `mul r/m`                                       | **无符号**乘。隐含 `rax`；结果在 `rdx:rax`      | CF OF             |
| `imul` | `imul r/m` / `imul r, r/m` / `imul r, r/m, imm` | **有符号**乘。双/三操作数形式更常用             | CF OF             |
| `div`  | `div r/m`                                       | **无符号**除。`rdx:rax ÷ r/m` → 商`rax` 余`rdx` | （未定义）        |
| `idiv` | `idiv r/m`                                      | **有符号**除。同上                              | （未定义）        |

> **除法陷阱**：`div`/`idiv` 前必须正确设置高位寄存器。32 位除法用 `cdq`（符号扩展 `eax`→`edx:eax`）或 `xor edx,edx`（无符号清零）；64 位用 `cqo`。除以 0 或商溢出会触发 `#DE` 异常。

**多精度加法示例（128 位 = 两个 64 位相加）：**

```nasm
; (rdx:rax) = (rsi:rdi) + (rcx:rbx)
add rax, rbx        ; 低 64 位相加，进位写入 CF
adc rdx, rcx        ; 高 64 位 + CF
```

## 3. 逻辑与位运算

| 指令   | 形式           | 说明                                                        | 影响标志          |
| ------ | -------------- | ----------------------------------------------------------- | ----------------- |
| `and`  | `and dst, src` | 按位与（常用于掩码、清零特定位）                            | ZF SF PF；CF=OF=0 |
| `or`   | `or dst, src`  | 按位或（置位特定位）                                        | ZF SF PF；CF=OF=0 |
| `xor`  | `xor dst, src` | 按位异或。`xor reg,reg` 是清零的惯用法（比 mov 0 更短更快） | ZF SF PF；CF=OF=0 |
| `not`  | `not r/m`      | 按位取反（一补码）。**不影响标志位**                        | —                 |
| `test` | `test a, b`    | 按位与但只更新标志、不写回（类比 `cmp` 之于 `sub`）         | ZF SF PF；CF=OF=0 |

> 惯用法：`test rax, rax` 检查 `rax` 是否为 0 / 正负（看 ZF/SF），等价于 `cmp rax, 0` 但更短。

## 4. 移位与循环移位

| 指令          | 形式                    | 说明                                                       |
| ------------- | ----------------------- | ---------------------------------------------------------- |
| `shl` / `sal` | `shl r/m, cl/imm`       | 逻辑/算术左移（二者相同）。低位补 0，移出位入 CF。等价 ×2ⁿ |
| `shr`         | `shr r/m, cl/imm`       | 逻辑右移。高位补 0（无符号 ÷2ⁿ）                           |
| `sar`         | `sar r/m, cl/imm`       | 算术右移。高位补**符号位**（有符号 ÷2ⁿ）                   |
| `rol`         | `rol r/m, cl/imm`       | 循环左移（移出位绕回低位）                                 |
| `ror`         | `ror r/m, cl/imm`       | 循环右移                                                   |
| `rcl`         | `rcl r/m, cl/imm`       | 带 CF 的循环左移（CF 参与轮转）                            |
| `rcr`         | `rcr r/m, cl/imm`       | 带 CF 的循环右移                                           |
| `shld`        | `shld dst, src, cl/imm` | 双精度左移（src 的高位移入 dst）                           |
| `shrd`        | `shrd dst, src, cl/imm` | 双精度右移                                                 |

> 移位计数只取低 6 位（64 位操作数）或低 5 位（32 位）。用变量移位时计数必须放在 `cl`。

## 5. 比较与测试

| 指令   | 形式        | 说明                                            |
| ------ | ----------- | ----------------------------------------------- |
| `cmp`  | `cmp a, b`  | 计算 `a - b` 只更新标志位。配合 Jcc 实现分支    |
| `test` | `test a, b` | 计算 `a & b` 只更新标志位。配合 Jcc 检查位/零值 |

**典型模式：**

```nasm
    cmp rax, rbx
    jl  .less          ; 有符号 rax < rbx 跳转
    ; ... rax >= rbx 的逻辑

    test rax, rax
    jz  .is_zero       ; rax == 0 跳转
```

## 6. 无条件跳转与调用

| 指令    | 形式              | 说明                                                        |
| ------- | ----------------- | ----------------------------------------------------------- |
| `jmp`   | `jmp label/r/m`   | 无条件跳转。支持直接、寄存器间接、内存间接                  |
| `call`  | `call label/r/m`  | 调用：将返回地址压栈后跳转                                  |
| `ret`   | `ret` / `ret imm` | 返回：弹出返回地址并跳转。`ret imm` 额外释放 imm 字节栈参数 |

| `enter` | `enter imm, 0`    | 建立栈帧（少用，编译器多手动展开）                          |

## 7. 条件跳转（Jcc）

> 全部基于 `cmp`/`test`/算术指令设置的标志位。同一行的多个名称是**同义词**（汇编后机器码相同）。

### 基于零标志

| 指令          | 条件   | 含义        |
| ------------- | ------ | ----------- |
| `je` / `jz`   | `ZF=1` | 相等 / 为零 |
| `jne` / `jnz` | `ZF=0` | 不等 / 非零 |

### 有符号比较

| 指令          | 条件            | 含义 |
| ------------- | --------------- | ---- |
| `jg` / `jnle` | `ZF=0 且 SF=OF` | `>`  |
| `jge` / `jnl` | `SF=OF`         | `>=` |
| `jl` / `jnge` | `SF≠OF`         | `<`  |
| `jle` / `jng` | `ZF=1 或 SF≠OF` | `<=` |

### 无符号比较

| 指令                  | 条件           | 含义                   |
| --------------------- | -------------- | ---------------------- |
| `ja` / `jnbe`         | `CF=0 且 ZF=0` | `>`（above）           |
| `jae` / `jnb` / `jnc` | `CF=0`         | `>=`（above or equal） |
| `jb` / `jnae` / `jc`  | `CF=1`         | `<`（below）           |
| `jbe` / `jna`         | `CF=1 或 ZF=1` | `<=`（below or equal） |

### 基于单个标志位

| 指令          | 条件    | 含义                      |
| ------------- | ------- | ------------------------- |
| `js`          | `SF=1`  | 结果为负                  |
| `jns`         | `SF=0`  | 结果非负                  |
| `jo`          | `OF=1`  | 有符号溢出                |
| `jno`         | `OF=0`  | 无溢出                    |
| `jc`          | `CF=1`  | 有进位/借位（无符号溢出） |
| `jnc`         | `CF=0`  | 无进位                    |
| `jp` / `jpe`  | `PF=1`  | 偶校验                    |
| `jnp` / `jpo` | `PF=0`  | 奇校验                    |
| `jrcxz`       | `rcx=0` | `rcx` 为零跳转（循环用）  |

## 8. 条件传送与置位（CMOVcc / SETcc）

**`CMOVcc`** —— 条件传送，按标志位决定是否传送。无分支，避免分支预测失败的开销。

```nasm
    cmp rax, rbx
    mov rcx, rax
    cmovl rcx, rbx     ; if (rax < rbx) rcx = rbx  → rcx = max(rax,rbx)
```

后缀同 Jcc：`cmove/cmovz`、`cmovne`、`cmovl/cmovg/cmovle/cmovge`、`cmovb/cmova/...`、`cmovs/cmovns`、`cmovc/cmovnc` 等。

**`SETcc`** —— 按条件把**一个字节**置 0 或 1（目标必须是 8 位 r/m）。

```nasm
    cmp rax, rbx
    setl al            ; al = (rax < rbx) ? 1 : 0
    movzx eax, al      ; 常配合零扩展为完整布尔值
```

后缀同 Jcc：`sete`、`setne`、`setl/setg/setle/setge`、`setb/seta/...`、`sets/setns` 等。

## 9. 栈操作

| 指令     | 形式           | 说明                                          |
| -------- | -------------- | --------------------------------------------- |
| `push`   | `push r/m/imm` | 压栈（`rsp -= 8` 后写入）。64 位下默认 8 字节 |
| `pop`    | `pop r/m`      | 出栈（读取后 `rsp += 8`）                     |
| `pushfq` | `pushfq`       | 压入 RFLAGS（64 位）                          |
| `popfq`  | `popfq`        | 弹出到 RFLAGS                                 |

> **栈对齐**：System V AMD64 ABI 要求 `call` 指令执行前 `rsp` 16 字节对齐。函数入口处因 `call` 压入返回地址，`rsp` 此时为 `16n+8`，故标准序言常用 `push rbp` 修正对齐。

## 10. 字符串指令

> 隐含使用 `rsi`（源）、`rdi`（目的）、`rcx`（计数）；方向由 `DF` 标志决定（`cld` 升序 / `std` 降序）。`rep` 系列前缀实现批量操作。

| 指令          | 说明                                                  |
| ------------- | ----------------------------------------------------- |
| `movsb/w/d/q` | 串传送：`[rdi] ← [rsi]`，指针按方向递增/减            |
| `stosb/w/d/q` | 串存储：`[rdi] ← al/ax/eax/rax`（常用于 memset）      |
| `lodsb/w/d/q` | 串加载：`al/ax/eax/rax ← [rsi]`                       |
| `cmpsb/w/d/q` | 串比较：比较 `[rsi]` 与 `[rdi]`，更新标志             |
| `scasb/w/d/q` | 串扫描：比较 `al/.../rax` 与 `[rdi]`（常用于 strlen） |
| `rep`         | 重复前缀：`rcx` 次（用于 movs/stos/lods）             |
| `repe/repz`   | 相等时重复（`rcx≠0 且 ZF=1`），用于 cmps/scas         |
| `repne/repnz` | 不等时重复（`rcx≠0 且 ZF=0`）                         |

**memset 示例（把 rcx 个字节设为 al）：**

```nasm
    cld
    mov rdi, buffer
    mov al, 0
    mov rcx, len
    rep stosb
```

## 11. 位扫描与位操作

| 指令     | 形式            | 说明                          |
| -------- | --------------- | ----------------------------- |
| `bt`     | `bt r/m, idx`   | 位测试：把第 idx 位读入 CF    |
| `bts`    | `bts r/m, idx`  | 位测试并置 1                  |
| `btr`    | `btr r/m, idx`  | 位测试并清 0                  |
| `btc`    | `btc r/m, idx`  | 位测试并取反                  |
| `bsf`    | `bsf r, r/m`    | 正向位扫描：最低置位位的索引  |
| `bsr`    | `bsr r, r/m`    | 反向位扫描：最高置位位的索引  |
| `popcnt` | `popcnt r, r/m` | 统计置位位的个数（人口计数）  |
| `lzcnt`  | `lzcnt r, r/m`  | 前导零计数（需 BMI/ABM 支持） |
| `tzcnt`  | `tzcnt r, r/m`  | 尾部零计数（需 BMI 支持）     |

> `bsf`/`bsr` 在操作数为 0 时结果未定义且置 `ZF=1`；`tzcnt`/`lzcnt` 对 0 有良好定义（返回操作数位宽），更推荐。

## 12. 符号/零扩展与类型转换

| 指令   | 作用                    | 等价高级语言       |
| ------ | ----------------------- | ------------------ |
| `cbw`  | `al` → `ax`（符号扩展） | int8 → int16       |
| `cwde` | `ax` → `eax`            | int16 → int32      |
| `cdqe` | `eax` → `rax`           | int32 → int64      |
| `cwd`  | `ax` → `dx:ax`          | 为 16 位 idiv 准备 |
| `cdq`  | `eax` → `edx:eax`       | 为 32 位 idiv 准备 |
| `cqo`  | `rax` → `rdx:rax`       | 为 64 位 idiv 准备 |

> 助记：`movzx`/`movsx` 用于**寄存器间**扩展；`cbw`/`cdqe`/`cqo` 这组是**累加器原地**扩展，主要服务于除法前的高位填充。

## 13. 系统与杂项

| 指令      | 说明                                                                                         |
| --------- | -------------------------------------------------------------------------------------------- |
| `syscall` | 进入内核（Linux x86-64 系统调用入口）。调用号放 `rax`，参数依次 `rdi, rsi, rdx, r10, r8, r9` |
| `nop`     | 空操作（占位、对齐、填充）                                                                   |
| `int imm` | 软中断（如老式 `int 0x80`，64 位下已被 `syscall` 取代）                                      |
| `cpuid`   | 查询 CPU 特性（输入 `eax`，结果在 `eax/ebx/ecx/edx`）                                        |
| `rdtsc`   | 读时间戳计数器到 `edx:eax`（性能测量）                                                       |
| `lock`    | 前缀：使后续内存指令原子化（用于 add/xchg/cmpxchg 等）                                       |
| `cmpxchg` | `cmpxchg r/m, r`：比较交换，无锁并发的基石（CAS）                                            |
| `xadd`    | `xadd r/m, r`：交换并相加（原子计数器）                                                      |
| `pause`   | 自旋锁中的提示指令，降低功耗与流水线压力                                                     |
| `ud2`     | 故意触发非法指令异常（标记不可达代码）                                                       |
| `hlt`     | 暂停 CPU 直到下一中断（特权指令）                                                            |

> **Linux syscall 约定要点**：返回值在 `rax`；`syscall` 会破坏 `rcx`（保存 rip）和 `r11`（保存 rflags）。这与普通函数调用约定不同，需特别注意。

## 14. 寻址模式

NASM 内存操作数统一写在方括号中：`[base + index*scale + disp]`

- **base**：任意 64 位通用寄存器
- **index**：除 `rsp` 外任意 64 位通用寄存器
- **scale**：`1`、`2`、`4`、`8`
- **disp**：8/32 位立即偏移

| 写法                 | 含义                                  |
| -------------------- | ------------------------------------- |
| `[rax]`              | 寄存器间接                            |
| `[rax + 8]`          | 基址 + 偏移                           |
| `[rax + rcx]`        | 基址 + 变址                           |
| `[rax + rcx*4]`      | 基址 + 比例变址（数组 `int[]` 访问）  |
| `[rax + rcx*8 + 16]` | 完整形式（结构体数组）                |
| `[rel symbol]`       | RIP 相对寻址（PIC，位置无关代码必备） |

> **大小限定**：当无法从寄存器推断操作数大小时，须显式标注：`mov byte [rax], 1`、`mov word [rax], 1`、`mov dword [rax], 1`、`mov qword [rax], 1`。

## 15. 寄存器一览

**64 位通用寄存器及其子寄存器：**

| 64 位      | 32 位        | 16 位        | 8 位（低）   | 调用约定角色（System V）                                          |
| ---------- | ------------ | ------------ | ------------ | ----------------------------------------------------------------- |
| `rax`      | `eax`        | `ax`         | `al`         | 返回值；除法被除数低位                                            |
| `rbx`      | `ebx`        | `bx`         | `bl`         | **被调用者保存**                                                  |
| `rcx`      | `ecx`        | `cx`         | `cl`         | 第 4 参数；移位计数                                               |
| `rdx`      | `edx`        | `dx`         | `dl`         | 第 3 参数；除法高位                                               |
| `rsi`      | `esi`        | `si`         | `sil`        | 第 2 参数                                                         |
| `rdi`      | `edi`        | `di`         | `dil`        | 第 1 参数                                                         |
| `rbp`      | `ebp`        | `bp`         | `bpl`        | **被调用者保存**；栈帧基址                                        |
| `rsp`      | `esp`        | `sp`         | `spl`        | 栈指针                                                            |
| `r8`–`r15` | `r8d`–`r15d` | `r8w`–`r15w` | `r8b`–`r15b` | `r8/r9`=第 5/6 参数；`r10–r11` 调用者保存，`r12–r15` 被调用者保存 |

**整数参数传递顺序**：`rdi, rsi, rdx, rcx, r8, r9`，超出部分压栈。
**调用者保存（caller-saved）**：`rax, rcx, rdx, rsi, rdi, r8–r11`。
**被调用者保存（callee-saved）**：`rbx, rbp, r12–r15`。

## 16. 标志位（RFLAGS）参考

| 位  | 缩写 | 名称             | 触发条件                                      |
| --- | ---- | ---------------- | --------------------------------------------- |
| 0   | `CF` | Carry            | 无符号运算进位/借位                           |
| 2   | `PF` | Parity           | 结果低 8 位中 1 的个数为偶                    |
| 4   | `AF` | Auxiliary Carry  | BCD 运算的半进位（bit3→bit4）                 |
| 6   | `ZF` | Zero             | 结果为 0                                      |
| 7   | `SF` | Sign             | 结果最高位为 1                                |
| 8   | `TF` | Trap             | 单步调试                                      |
| 9   | `IF` | Interrupt Enable | 是否响应可屏蔽中断                            |
| 10  | `DF` | Direction        | 字符串指令方向（`cld`=0 升序 / `std`=1 降序） |
| 11  | `OF` | Overflow         | 有符号运算溢出                                |

## 附录：最小可运行模板（Linux x86-64）

```nasm
; 编译运行：
;   nasm -f elf64 hello.asm -o hello.o
;   ld hello.o -o hello
;   ./hello

section .data
    msg     db  "Hello, NASM!", 10   ; 10 = '\n'
    msglen  equ $ - msg              ; 当前地址 - msg 起始 = 长度

section .text
    global _start

_start:
    ; write(1, msg, msglen)
    mov rax, 1          ; sys_write
    mov rdi, 1          ; fd = stdout
    mov rsi, msg        ; buf
    mov rdx, msglen     ; count
    syscall

    ; exit(0)
    mov rax, 60         ; sys_exit
    xor edi, edi        ; status = 0
    syscall
```
